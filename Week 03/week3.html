  • Chapter 5: Objects

      Object Literals

      • An object is a self-contained set of related values and functions
      • If a properties value is a function known as a method
      • Objects are often used to keep related information and functions grouped together

      Creating Objects

      • To create an object you use {}
      • To create a constructor object you have to use the keywords new Object()
      • You can assign an object to a variable like this: const spiderman = new Object();
      • You can access an object's properties by doing either spiderman.property or spiderman['property']

      Calling Methods

      • Calling an object method, you use the dot or bracket method
      • Calling a method is the same as calling a function
      • You can check if a property or method exits in an object by using in, ex: 'city' in superman; this may return false or true depending on if the method or property exists
      • You can loop trough all the properties of a object by using the statement for in

      Object Modification

      • You can add an property by using the dot method like so, superman.city = 'Metropolis'
      • You can modify properties like so, superman['real name'] = 'Kal-El'
      • You can remove a porperty by using the keyword delete, delete superman.fly
      • You can nest objects in another object and then access by adding onto the previous dot or bracket statement, jla.flash.realName
      • You can mix different notation like this, jla.aquaman['realName']
      • You can clone objects like this, const cloneThor = thor;

      Chapter 6: The Document Object Model

      The DOM

      • The DOM allows you to access parts of the web page and allow you to interact by modifying elements
      • The DOM treats everything on the web page as a node on a tree
      • The DOM is not apart of JavaScript but can be used with any programming language, though JavaScript has a built in object to interact called document

      History of DOM

      • The web has developed along th way and the method to interact was standardized by W3C to create the DOM level 1 which would eventually be updated to the current DOM

      Getting Elements

      • To access pieces of the the page you can use the dot method like so, doucment.body
      • If you are curious what type of node it is you can use the following dot method on any part of the page, body.nodeType;
      • Doing that will return a number which signifies what type of node it is
      • The numbers represent the following:
        1, An element
        2, An attribute
        3, Text
        8, A comment
        9, Body
      • You can use the following to find the node type name, body.nodeName;
