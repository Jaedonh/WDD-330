  • Chapter 10 - Errors, Exceptions, and Warnings

    • Errors are caused when something goes wrong in the code
    • Errors are usually caused by three types of things
      1. System Errors, a problem with the system or external devices
      2. Programmer Errors, program contains invalid syntax, typos, or faulty logic
      3. User Error, user has entered data in incorrectly which program cannot handle
